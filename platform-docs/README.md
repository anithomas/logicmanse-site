# logicmanse.ca — Bookkeeping Platform

Multi-tenant bookkeeping platform. See CLAUDE.md for standing rules and docs/bookkeeping-master-document.md for the full spec.

## Getting started
1. Open this folder in a terminal
2. Run `claude` to start Claude Code (it auto-reads CLAUDE.md)
3. Paste the contents of docs/first-prompt.md as your first message
